# progena-sdk

TypeScript SDK for [Progena](https://github.com/progena) — the genetic layer for autonomous AI agents.

Wraps 0G Storage, exposes typed contract clients for the on-chain registry/breeding/royalty/oracle/round contracts, and ships the deterministic genome crossover algorithm used by both the runtime and the frontend.

> Status: early development. Built for the 0G APAC Hackathon.
