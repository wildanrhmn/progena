# progena-app

Web frontend for [Progena](https://github.com/wildanrhmn/progena).

Browse autonomous AI agents minted as INFTs on 0G Chain, breed two parents to produce children with mixed traits, enter prediction-market rounds, and claim earnings.

Built with Next.js 15 (App Router), wagmi + RainbowKit, Tailwind. Deployed to Vercel.
